# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full text is at the link above. We do not fork it.

## Scope

The Covenant applies to all project spaces — the repository, issue tracker, pull requests, discussions, and any public communication channel where someone is acting as a representative of FlashPaste.

## Reporting

If you believe someone is acting in violation of the Code of Conduct, email the maintainer at **<webubusiness@gmail.com>** with subject `flashpaste conduct`. Reports are handled in confidence. The maintainer will acknowledge receipt within 72 hours and respond with a resolution or update within 14 days.

Anyone may report. You do not need to be the target of the behaviour.

## Enforcement

The maintainer enforces the Covenant. Consequences scale with severity and recurrence, following the Covenant's "Enforcement Guidelines" ladder (correction → warning → temporary ban → permanent ban). The maintainer's decision is final, with one appeal permitted via the same reporting email.
